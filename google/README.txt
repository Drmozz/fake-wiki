
Fake Google Magie
-----------------
- index.html : page principale à ouvrir
- barre-google-search.jpg : image de la barre Google mobile (en haut)
- barre-safari-bottom.jpg : fausse barre Safari (en bas)
- firstpagegoogle.jpg : page d’accueil Google mobile (splash écran)
- manifest.json : support plein écran (PWA)
- icon192.png : icône pour "ajouter à l’écran d’accueil"
- Utilise la même configuration Firestore et logique que la page fake wiki

Instructions :
- Place toutes les images dans le même dossier que index.html (ou adapte les chemins dans le code si besoin)
- Ouvre index.html sur Netlify/serveur local, fonctionne en PWA sur iOS/Android
